Privacy Policy

Effective Date: 30.10.2025

This Chrome extension CaganT's Homepage is developed to enhance your browser experience by providing useful features. We respect your privacy and are committed to protecting your personal information. This Privacy Policy explains how we collect, use, store and share data.

1. Data Collection

The Extension does not collect personally identifiable information unless explicitly stated and shown to the user. The Extension may access or store the following types of non-personal data to operate correctly:

  • User preferences: settings or layout choices you make inside the Extension (stored locally).
  • Local storage data: data saved using Chrome Storage APIs or localStorage to persist preferences and state.
  • Anonymous diagnostic data (optional): minimal information for debugging or improving reliability (for example, version number, error codes). This data is not tied to any identifiable user.

2. How We Use Data

Any data collected is used only to deliver, maintain and improve the Extension's features. Examples:

  • Restore your settings when you reopen your browser.
  • Fix bugs and improve performance when anonymous diagnostic data is provided.

3. Data Storage

By default, data is stored locally on your device using Chrome Storage APIs or localStorage. The Extension does not send personal data to external servers unless you explicitly enable a feature that requires it (for example, an optional sync or cloud backup feature).

If you enable any feature that transmits data off-device, we will clearly disclose what is transmitted and ask for your consent before doing so.

4. Data Sharing & Third Parties

The Extension does not share personal or sensitive user information with third parties, advertisers, or analytics providers. If we ever need to share data (for example, with a service provider for an optional feature), we will:

  • Explain what data is shared and why.
  • Only share the minimum required information.
  • Require the third party to protect the data and not to sell it.

5. User Control

You can control your data in the following ways:

  • Remove or change settings inside the Extension's options page.
  • Clear browser storage or uninstall the Extension to delete locally stored data.
  • If an optional cloud feature exists, you can disable it at any time to stop transmission of data to remote servers.

6. Security

We implement reasonable measures to protect stored data from unauthorized access or disclosure. However, no system is completely secure — please avoid storing extremely sensitive personal information in the Extension.

7. Children's Privacy

The Extension is not directed to children under 13. We do not knowingly collect personal information from children under 13.

8. Changes to This Policy

We may update this Privacy Policy periodically. When we do, we will post the updated policy and update the "Effective Date" above.
